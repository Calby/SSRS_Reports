# Published SSRS Reports

Source-controlled .rdl files for all deployed reports.
Each file here should match what is live on the report server.

## Rules
- Always update the .rdl file here BEFORE deploying a change to the server
- File name must match exactly what is deployed on the report server
- After deploying, verify the report runs correctly on the server
- Update docs/report-index.md with the Last Updated date after every deployment
